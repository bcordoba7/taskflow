#taskflow

##Descripcion del proyecto:
TaskFlow es una aplicacion de consola desarrollada en C# para gestionar y automatizar tareas internas del equipo, con una estructura simple y escalable para futuras mejoras.

##Como ejecutarlo:
1. Instalar el SDK de .NET
2. Abrir una terminal en la raiz del proyecto
3. Ejecutar:

dotnet restore
dotnet run --project src/TaskFlow

# Tecnologias usadas
- C#
- .NET
- GIT
- GitHub
